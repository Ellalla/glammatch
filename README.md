# GlamMatch

GlamMatch 是一个帮助用户寻找附近美妆艺术家的移动应用。

## 功能特点

- 优雅的用户界面设计
- 响应式布局
- 现代化的 UI 组件

## 技术栈

- React Native
- Expo
- NativeWind (Tailwind CSS for React Native)
- React Native Web

## 开始使用

1. 克隆仓库
```bash
git clone https://github.com/yourusername/glammatch-app.git
```

2. 安装依赖
```bash
cd glammatch-app
npm install
```

3. 启动开发服务器
```bash
npm start
```

## 开发环境要求

- Node.js >= 14
- npm >= 6
- Expo CLI
